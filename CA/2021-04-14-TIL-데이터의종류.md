#### 진법과 진법변환
- **디지털 정보의 단위**
    - 1nibble = 4bit
    - 1byte = 8bit
    - 1byte = 1문자
    - 영어는 1byte, 한글은 2byte
    - 1word : 특정 CPU에서 취급하는 명령어나 데이터의 길이에 해당하는 비트 수

- **보수**

    - 1진 보수
      최대값을 형성하는데 서로 보완 관계에 있는 두 수 사이의 관계

    - 2진 보수
      최대 표현 자리 수를 형성하는데 서로 보완관계에 있는 두 수 사이의 관계

eg) 10진수에서의 보수
- A+B = 9, A+B = 99 => 1진 보수
- A+B = 10, A+B = 100 => 2진 보수

eg) 2진수에서의 보수
- a+b = 1, a+b = 11, a+b = 111 => 1진 보수
- a+b = 10, a+b = 100, a+b = 1000 => 2진 보수
